# 🐦 Flappy Bird (Java)

A simple Flappy Bird game built using Java Swing and AWT.

## 🎮 Features

* Bird movement with gravity
* Pipes generation
* Collision detection
* Sound effects

## ▶️ How to Run

1. Compile:
   javac App.java FlappyBird.java
2. Run:
   java App

## 📁 Files

* App.java
* FlappyBird.java
* Images & Audio files
